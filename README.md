# MoodMarket

MoodMarket is a local-first MVP for an emotional consumption mirror. It helps users examine whether a purchase is grounded in need or pulled by mood, identity, social pressure, urgency, or algorithmic exposure.

Open `index.html` in a browser to run it. The app is a static single-page experience with hash navigation and browser-local persistence.

## Implemented Surface

- Mirror: a purchase dossier, live scoring engine, guardrail checks, cooldown/pass/save actions, and copyable readouts.
- Library: persisted decision records with search, filters, status changes, reopen, and delete.
- Patterns: aggregate trigger charts, average signal charts, avoided-spend totals, bought-spend totals, and a pattern brief.
- Protocols: configurable monthly limit, cooling period, risk ceiling, price review threshold, personal values, active rules, JSON export, and JSON import.
